Functions

JAVASCRIPT

En JavaScript, una función es un bloque de código reutilizable que sirve para realizar una tarea específica. Podés definirla una vez y ejecutarla (llamarla) todas las veces que quieras.

  • Evitar repetir código
  • Organizar mejor el programa
  • Recibir datos (parámetros)
  • Devolver un resultado (return)
