Spread Operator

JAVASCRIPT

El spread operator que incorpora ECMAScript 6 en JavaScript es un operador que simplifica la recogida de valores en una estructura de datos. Su representa con tres puntos: ...

Volver al menu 
        
    const numeros = [4,5,6,27,18];
    let numeroscopiados = [...numeros];
    numeroscopiados[0] = 464;
    console.log (numeros);
    console.log (numeroscopiados);

    const nombres = ["Juan", "Martin", "Diego", "Gabriel"];
    const otrosnombres = ["Pedro", "Alejandro", "Ezequiel", "Ignacio"];
    console.log (...nombres, ...otrosnombres, "Ruben");
