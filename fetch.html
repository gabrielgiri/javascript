JAVASCRIPT

fetch es la API moderna de JavaScript para hacer peticiones HTTP (traer o enviar datos a un servidor). Funciona de forma asíncrona y devuelve promesas.

  • fetch() realiza una petición HTTP
  • Promise fetch devuelve una promesa
  • async / await permite manejar código asíncrono
  • Response objeto que representa la respuesta del servidor
  • response.ok indica si la petición fue exitosa
  • response.json() convierte la respuesta a JSON
  • try / catch maneja errores de la petición
⬅ Volver al menú

Ejemplo con usuarios

Consola:

async function getUsers() {
  try {
    const response = await fetch("https://api.example.com/users");

    if (!response.ok) {
      throw new Error("Error HTTP: " + response.status);
    }

    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("Error:", error);
  }
}

Ejemplo con movies

Consola:

async function getMovies() {
  try {
    const response = await fetch(
      "https://jsonfakery.com/movies/paginated"
    );

    if (!response.ok) {
      throw new Error("Error HTTP: " + response.status);
    }

    const data = await response.json();

    const console2 = document.getElementById("console2");
    console2.innerHTML = "";

    data.data.forEach((movie) => {
      const div = document.createElement("div");
      div.textContent = movie.original_title;
      console2.appendChild(div);
    });
  } catch (error) {
    console.error("Error:", error);
  }
}

getMovies();
