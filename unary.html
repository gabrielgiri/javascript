Unary

JAVASCRIPT

En JavaScript, Unary se refiere a los operadores unarios, es decir, operadores que trabajan con un solo operando.

Volver al menu 

    let year = 2021;
    console.log(year)    // 2021
    console.log(year++)  // 2021
    console.log(year)    // 2022

    let birth = "1973";
    console.log(typeof(birth))    // string
    console.log(typeof(+birth))   // number
