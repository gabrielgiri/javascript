Deleting Properties

JAVASCRIPT

⬅ Volver al menú

En JavaScript, delete solo funciona para borrar propiedades de objetos. No borra variables declaradas en el ámbito actual, a menos que sean propiedades del objeto global. 


var fechas = { a: 1985, b: 1980, c: 1974, d: 2001 },
    myVar = 10;

delete fechas.a;   // ✔ Funciona, elimina "a"
delete myVar;      // ❌ NO funciona: las variables no pueden borrarse

console.log(fechas);
console.log(myVar);
