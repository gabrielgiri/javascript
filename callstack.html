Call Stack

JAVASCRIPT

Es una estructura de datos donde JavaScript guarda qué función se está ejecutando en cada momento.

  • Call: Llamada
  • Stack: Pila
      • LIFO
    • LI: Último en entrar
    • FO: Primero en salir

Código del tutorial


// Funciones demostrativas del Call Stack
function a() {
  log("Entró en a()");
  b();
  log("Saliendo de a()");
}

function b() {
  log("Entró en b()");
  c();
  log("Saliendo de b()");
}

function c() {
  log("Entró en c()");
  log("Función c ejecutada");
  log("Saliendo de c()");
}

a(); // Inicio
