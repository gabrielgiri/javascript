SCOPE

JAVASCRIPT

Volver al menu 
             
   scope global
   Las variables globales pueden ser accedidas desde cuaquier parte del programa.
   Son declaradas fuera de una funcion o bloque de codigo
   scope local
   Las variables globales pueden ser accedidas desde una parte del programa
   Codigo
let texto = "Gabriel";

function llamado() {
    let texto = "Alejandro";

    function mostrarNombre() {
        console.log('Yo soy ' + texto);
    }

    mostrarNombre();
}

llamado();
